# Shell basics

show hidden files
`ls -a`

more info on files
`ls-la`

create folder
`mkdir`

go into folder
`cd`

print working directory (see where you are)
`pwd`

create file
`touch`

remove files (delete forever)
`rm`

show contents of file
`cat`

to remove whole folder - remove recursive
`rm -r` 

the folder you are in
`./`

one folder higher
`../`
